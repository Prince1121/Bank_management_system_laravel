# Online Banking

An Online Banking Platform Built With Laravel


## Usage

Note: Require composer

Installation
In the command terminal, run the following commands:

``` bash


$ cd Online-banking/
$ composer install
$ php artisan migrate

```

Open the .env file and set your database credentials

``` bash
DB_DATABASE=bank
DB_USERNAME=root
DB_PASSWORD=

```
Then run the migration command

``` bash

$ php artisan migrate

```
